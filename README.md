# Generator

### Prerequisites

NodeJS 6+
 
MongoDB
 
Gulp `npm install -g gulp-cli`
 
Bower `npm install -g bower`

### Install


```sh
git clone https://github.com/ReeganExE/exergentor.git exergentor
cd exergentor
npm install
bower install
```

### Usage
Start MongoDB, then in the `exergentor`, run:
```sh
gulp
```
